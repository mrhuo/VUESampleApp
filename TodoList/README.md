# Todolist App
使用 `vue.js` 开发的 `Todolist` 实例。
在线预览: http://res.mrhuo.com/demo/todolist/index.html

#### 特性:
-----------------------------------
* 基于 `bootstrap` 的自适应布局
* 数据保存于 `localStorage`
